Custom AAA                                                                                
~~~~~~~~~~                                                                                
                                                                                          
The ``secure_cisco_ios_custom_aaa`` variable can be used to define                        
custom AAA settings. This will also prevent TACACS+ and Radius settimgs                   
from making their own AAA adjustments.                                                    
                                                                                          
.. code-block:: yaml                                                                      
                                                                                          
    secure_cisco_ios_custom_aaa: []
